About Me

The Developer Behind the Code

I'm a Frontend Engineer based in Gurugram, specialising in React.js, JavaScript, HTML5, CSS3, and SCSS. I build the kind of interfaces that real enterprise teams depend on daily — dashboards, data grids, component systems, and accessibility-compliant UIs that work at scale.

My day-to-day has involved building features across coaching, QA management, knowledge management, real-time analytics dashboards, and the internal UI component library.

I take pride in writing UI that is fast, accessible, and maintainable. I'm currently deepening my expertise in React advanced patterns, Redux, Jest testing, and DSA — because good developers never stop building.
